## Features

- **Industrial Aesthetic** — Design language inspired by manufacturing contexts: safety markings, machine surfaces, and technical documentation
- **Monospace Typography** — Clean, technical typography that fits the industrial theme
- **Dark Mode** — Automatic dark mode support with carefully tuned colors
- **Zero Runtime** — Pure CSS styling with data attributes, no JavaScript overhead
- **TypeScript First** — Full type definitions with JSDoc documentation
- **Accessible** — Built with accessibility in mind, proper focus states and ARIA support
