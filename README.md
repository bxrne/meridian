# meridian

<img src="./demo.png" alt="Meridian Demo" width="360"/>

contemporary industrial style component library

## Installation

```bash
bun add meridian
```

## Usage

```tsx
import 'meridian/styles.css';
import { Button } from 'meridian';

function App() {
  return <Button>Click me</Button>;
}
```
